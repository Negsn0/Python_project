## 8. 最適化と収束

### 絶対値と最小値

```Python
# 絶対値の計算
temp = abs(minimum - K)  # 2つの値の差の絶対値

# 最小値の更新
if temp < minimum:
    minimum = temp  # より小さい値で更新
```

### 剰余と最適化

```Python
# 剰余の計算と最適化
r = N % K  # NをKで割った余り
print(min(r, abs(r - K)))  # 余りと(K-余り)の小さい方を出力
```
